=========================================
C-BRIDGE MEMORY LEDGER
Track all data structures crossing into C
=========================================

1. CORPUS
   - Original Python: 2D List of sentences (Strings)
   - Intermediate Python: 2D List of sentences (Integer IDs)
   - C-Ready Format: 1D numpy array (np.int32)
   - C Pointer: int*
   - Note: Injects '-1' as the EndOfSentence delimiter.
   - Reversal: Split the 1D array at every '-1', then map IDs back to words using id_to_word.

2. DISCARD PROBABILITIES
   - Original Python: Dictionary {word_str: float_prob}
   - C-Ready Format: 1D numpy array (np.float32)
   - C Pointer: float*
   - Note: The array index matches the word ID exactly. 
   - Reversal: Loop through the array, pair the index with the probability, and map index -> word via id_to_word.

3. UNIGRAM TABLE (Negative Sampling)
   - Original Python: 1D List of Integer IDs
   - C-Ready Format: 1D numpy array (np.int32)
   - C Pointer: int*
   - Note: Array size is typically 10,000,000. 
   - Reversal: Not typically needed, purely used for RNG lookups.

4. TARGET MATRIX (V)
   - Original Python: 2D numpy array (np.float32)
   - C-Ready Format: 1D flattened numpy array (np.float32)
   - C Pointer: float*
   - Note: Handled automatically by ctypes/numpy.

5. CONTEXT MATRIX (U)
   - Original Python: 2D numpy array (np.float32)
   - C-Ready Format: 1D flattened numpy array (np.float32)
   - C Pointer: float*
   - Note: Handled automatically by ctypes/numpy.

def decode_corpus_to_text(c_corpus_array, id_to_word):
    sentences = []
    current_sentence = []
    for word_id in c_corpus_array:
        if word_id == -1:
            if current_sentence:
                sentences.append(current_sentence)
            current_sentence = []
        else:
            current_sentence.append(id_to_word[word_id])
    return sentences

def decode_probs_to_dict(c_probs_array, id_to_word):
    return {id_to_word[i]: prob for i, prob in enumerate(c_probs_array)}
